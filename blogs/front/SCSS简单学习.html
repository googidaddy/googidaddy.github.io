# 混合

对经常重复使用的代码，使用 mixin 可以“打包”，然后随处用

# 定义-mixin

@mixin clearfix {
	&:after {
		content: '';
		display: block;
		clear: both;
	}
}

# 继承-%

# 不同

与混合不一样的是，混合编译为 CSS 后，会在每个引入混合样式的选择器下插入样式，而继承则是把继承的选择器加入到共有的样式选择器前，不会被编译到 CSS 文件中。
