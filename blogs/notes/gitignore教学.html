gitignore教学

sicko2021-07-01 00:00:00notesgitignore

.gitignore是一个好东西

.gitignore文件存在是为了告诉git在上传代码时，要忽略哪些文件

Q：我写了规则在里面，怎么上传的时候没有生效

A：git缓存需要清除

# solution
git rm -r --cached .
git add .
git commit -m "update .gitignore"
