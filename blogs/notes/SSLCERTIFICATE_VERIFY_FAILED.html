服务器安装ssl证书

sicko2021-03-20 00:00:00notescentosSSLubuntu

近期弄了一点爬虫并且部署到了服务器镜像里，但是镜像竟然报错了，于是乎折腾了5分钟（宝贵的生命X

(caused by URLError(SSLCertVerificationError(1, '[SSL: CERTIFICATE_VERIFY_FAILED] certificate verify failed: unable to get local issuer certificate (_ssl.c:1108)')))
1

解决办法:

# Linux

请搜索linux install certificates.command

我的Ubuntuapt-get install install ca-certificates

centos yum install ca-certificates

# windows

请搜索(最好是pycharm)

windows install certificates.command

pycharm可以在目录下一键安装
