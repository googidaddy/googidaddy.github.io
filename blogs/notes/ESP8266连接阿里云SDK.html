# ESP8266 连接阿里云

最近在倒腾 ESP8266，弄物联网，为了联上阿里云物联网平台，得多加使用搜索引擎。

第一步先 Arduino 适配合适的板子(esp8266)和型号，具体自己搜一下，这个要学会自己配置

Arduino IDE 配置

  • 首选项 - 附加开发板管理器网址 输入 https://arduino.esp8266.com/stable/package_esp8266com_index.json
  • 工具 - 开发板 - 开发板管理器 - 搜索 8266 并安装

为了避免大家走弯路，直接给出合理官方的 SDK。（切记那些 18、19 年的教学早已过时）

Arduino-Aliyun-IoT-SDK，这个是 Arduino 的阿里 IOT 库，官方仓库。

使用上照着仓库 Usage，安装好 ArduinoJson,Crypto,PubSubClient 库，根据使用实例填好所需项。

然后注意！打开 Arduino 的 libraries 的 PubSubClient 的 Src 文件夹，找到PubSubClient.h修改 MQTT_MAX_PACKET_SIZE 和 MQTT_KEEPALIVE 的值，将其改大点，比如改成 1024 和 60。

感谢填坑者！

最后连接。
