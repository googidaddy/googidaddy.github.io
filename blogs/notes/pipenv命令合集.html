pipenv命令合集。

# pipenv指南

# 特点

  • 不需要再维护requirements.txt，用pipfile和pipfile.lock
  • 兼容多个python版本

# 安装

pip install pipenv
1

# 常用命令

# 安装包
pipenv install
# pipenv install flask

# 激活当前项目的虚拟环境
pipenv shell

# 安装开发依赖包
pipenv install pytest --dev

# 图形显示包依赖关系
pipenv graph

# 生成lockfile
pipenv lock

# 删除所有的安装包
pipenv uninstall --all
