制作镜像抛出/bin/sh: apt-get: not found?

# Building image throws “/bin/sh: apt-get: not found”

这是我第一次实战Dockerfile的时候遇到的问题。

经过搜索，美国知乎给我一个相对于很好的答案

# 坑1️⃣

安装东西的时候主要要带-y全局yes以防卡住中断

apt-get install -y
1

# 坑2️⃣

没指定工作目标，第一次写，难免要加载目录的requirements，然后not found

WorkDIR指定工作目录，以防迷路

FROM XXX
COPY ./XX /XX
WORKDIR /XX
RUN XXX
1
2
3
4

# 坑3️⃣

没暴露端口EXPOSE

EXPOSE 80
