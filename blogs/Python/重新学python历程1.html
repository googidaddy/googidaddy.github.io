# 重新学python历程-1

至于为什么叫重新学python，是因为之前在校期间，仅是做了几个程序demo就撒手了，应当补回来。

网上教的通常都是解决案例，图一贴，现成直接抄。但是对于新手来说，这并不是你该有的开局。

首先得认清楚你学python是为了什么，程序是为了什么目的而服务，当你的视野开拓后，想必学起来心里有底。
