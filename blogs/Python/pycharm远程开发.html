pycharm远程开发

sicko2021-07-01 00:00:00Pythonpycharm远程开发python

远程开发真的太好了

# 简介

一般本地作为编写代码，需要调试或者运行代码时使用远程服务器的环境，电脑负荷少，是一种享受

原理：利用SSH链接远程服务器交互数据

流程

  • 在远程服务器创建docker container
  • 在远程服务器配置ssh
  • Pycharm链接远程服务器
  • Pycharm链接远程服务器的container

# 配置docker

安装好docker并且下载好相应的image，配置容器端口映射,这里使用的是8022映射22

-p 8022:22

# 远程服务器ssh配置

需要在远程服务器和远程服务器上的docker container都要安装上openssh

apt update && apt install -y openssh-server
1

配置ssh服务

# 操作均在docker container中
chpasswd
# 这里使用你自己设置密码，但是一定要记住！(两次输入)

# 这个命令是运行root身份登录
# 记得打开/etc/ssh/sshd_config
# 查看是否有PermitRootLogin yes (未被注释的)
sed -i 's/PermitRootLogin prohibit-password/PermitRootLogin yes/' /etc/ssh/sshd_config

sed 's@session\s*required\s*pam_loginuid.so@session optional pam_loginuid.so@g' -i /etc/pam.d/sshd

echo "export VISIBLE=now" >> /etc/profile
