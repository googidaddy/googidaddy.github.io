# 增加docker-compose

使用docker-compose是一个 run container的一个更好替换，像配置nginx反向代理和api服务，应该隔离开两个容器。

version: "3.9"
services:
  web:
    build: nginx
    ports:
      - 8080:8080

  api:
    build: Docker-main
    ports:
    - 9090:9090
