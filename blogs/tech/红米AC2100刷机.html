红米AC2100刷机

sicko2021-04-05 00:00:00tech红米AC2100刷机

从恩山那里学习总结过来的，单刀直入。感谢大大

# 前期准备

红米AC2100-2.0.7固件

Padavan固件-下载redmi211.trx

XShell（自行下载）

如果说你有比较好的老毛子(Padavan)固件什么的，可以自行替代

红米ac2100闪存分为晶豪（欧皇）和东芝两种，一般有坏块3-4个（不是太多）也无妨，刷机代码已经规避了

本人到手也是是东芝，检测有坏块，然后去换了，成功到手晶豪

# 开始

综合步骤如果看不懂可以参考此视频

首先登录进路由器管理地址

点系统状态，手动升级，升级红米ac2100-2.0.7固件

等待升级重启完毕，路由器灯会变成正常的蓝色，再次打开路由器管理地址

此时地址会变为http://192.168.31.1/cgi-bin/luci/;stok=<STOK>/web/home#router

复制下来<STOK>

替换stokhttp://192.168.31.1/cgi-bin/luci/;stok=<STOK>/api/misystem/set_config_iotdev?bssid=Xiaomi&user_id=longdike&ssid=-h%3B%20nvram%20set%20ssh_en%3D1%3B%20nvram%20commit%3B%20sed%20-i%20's%2Fchannel%3D.*%2Fchannel%3D%5C%22debug%5C%22%2Fg'%20%2Fetc%2Finit.d%2Fdropbear%3B%20%2Fetc%2Finit.d%2Fdropbear%20start%3B

然后将网址复制下来粘贴网页打开，会显示"code":0代表成功

打开xshell，连接192.168.31.1 用户名为root 密码admin

直接输入一下命令刷入Breed，并等待20s

cd /tmp && wget https://breed.hackpascal.net/breed-mt7621-xiaomi-r3g.bin && mtd -r write breed-mt7621-xiaomi-r3g.bin Bootloader

或者下面命令：
cd /tmp && wget http://openwrt.ink:8666/Breed/breed-mt7621-xiaomi-r3g.bin && mtd -r write breed-mt7621-xiaomi-r3g.bin Bootloader

正常结束后是会显示reboot，然后断开与ssh连接，（自己尝试能不能输入东西就知道了）

路由器断电，找个针戳住路由器背后的reset小孔并插电，记得是戳住，等路由器灯一直闪烁后，浏览器输入192.168.1.1进入breed

找到环境变量，增加:xiaomi.r3g.bootfw 值：2 然后保存

点击更新固件，上传padavan固件包，等待更新完成，然后就可以进入padavan路由器管理界面

开启5G频道->160HZ，冲冲冲
