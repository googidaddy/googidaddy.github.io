国行PS5配置代理

sicko2021-06-07 00:00:00tech国行PS5翻墙clash

解决在宿舍不可用路由器、不会刷路由器、不想买主机盒子，同时有笔记本但想国行PS5上外网的方法。

# set up

自备机场，安装好clash，添加好你的clash节点profile（一般机场自带有订阅）

Allow LAN 开启，记录以太网IP地址和7890端口（如果你改了，请更换，一般默认不会改）

打开电脑开启热点，PS5连接该热点WiFi，在网络详细设定中，设置Proxy，把以太网IP和端口填上，

# 验证

打开Twitch，如果能看直播，那就代表成功了。失败请检查自己的步骤。
