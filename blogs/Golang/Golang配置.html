# install

下载地址(根据自己情况匹配系统)-->我是windows

安装地址我放在了装Golang的同目录下（默认地址也行）D:\jetbrains\Goland

不用配置系统Path，重复造轮子还没造够？

# setting

打开Golang，new a project

在项目下新建一个bin和src文件夹，src里新建一个go file

file里代码如下 just a demo

// go运行需要一个启动入口，启动入口包名需要是main
package main

import (
	"fmt"
)

func main()  {
	fmt.Println("Hello World")
}

# Run

显然我们是要让一个file.go运行起来，所以需要右上角配置一下环境

尽可能不长篇大论解释，越啰嗦脑子越多大便

GoRoot = Python编译程序

GoPath = Project目录

项目下一般有3个文件夹：bin(放编译后的exe文件)、pkg（放自定义的包）、src（放源文件）

pkg等后面了解了会补充的

# summary

不希望大家浪费过多时间在环境配置上面,1分钟下载完，1分钟创建所需文件，1分钟配置环境

VS CODE的话，比较复杂一点，每一个project，都需要用终端配置Go工作区
