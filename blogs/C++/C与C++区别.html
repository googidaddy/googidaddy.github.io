# 提前看结论

C++完全兼容C语言，并且拥有许多新功能

# 指针与应用

C++中兼容了C的指针，但是对于变量传递而言，更推荐使用C++的引用。因为引用指向的对象不能为空，因此可以避免一些空地址访问问题。

# struct

C和C++都有这一个结构，而C++中的struct是一种类，它与类唯一的区别是它的成员和继承行为默认是public的，而一般的默认成员是private的。

声明struct时两者都有区别,C版本:

typedef struct Node_t{
    struct Node_t *next;
    int key;
} Node;
