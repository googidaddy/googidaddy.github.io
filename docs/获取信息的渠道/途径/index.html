base​

很久之前获取信息都是通过单一搜索相关技术问题，然后靠搜索引擎提供的相关个人博客来获取信息开拓视野，再之后就是通过一些论坛V2之类的消息渠道，现在是定制个人rss，不断缩小范围，精准获取干货范围。
