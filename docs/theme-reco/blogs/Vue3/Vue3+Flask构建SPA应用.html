Vue3+Flask构建SPA应用

sicko 2021-03-01 Vue3FlaskaxiosElement-plus

快速搭建一个简单的由Vue3提供的frontend和flask提供后端的SPA(单页面应用)，用vscode即可完成，有任何问题可以邮箱或者其他联系方式提问，评论区等有空再弄。

# 安装所需环境

  1. 用Vite初始化Vue3

    npm:

    $ npm init @vitejs/app 你的项目名字
$ cd 你的项目名字
$ npm install
# 下面这一步仅测试
$ npm run dev

  2. 添加Element-plus作为组件UI，引用方法为全局，添加axios。

    npm:

    npm install element-plus --save
npm install axios -S

    2

    在main.js中写入以下内容

    import { createApp } from 'vue'
import ElementPlus from 'element-plus';
import 'element-plus/lib/theme-chalk/index.css';
import axios from "axios";
import App from './App.vue';

const app = createApp(App)
app.config.globalProperties.$axios = axios;
app.use(ElementPlus)
app.mount('#app')

 methods: {
    onSubmit() {
        // 全局引用axios
      this.$axios
        // 发送post请求到flask建立的api
        .post("http://127.0.0.1:5000/upload")
        .then((res) => {
          // 把flask的响应数据通过控制台log出来
          console.log(res.data);
          // 弹窗success
          ElMessage.success({
            message: "恭喜你，这是一条成功消息" + res.data,
            type: "success",
          });
        });
    },
  },
