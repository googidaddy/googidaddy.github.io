centos7安装python3.9

sicko 2021-02-16 centospython

阅读本文大概需要4分钟

如果真的条件只有centos系统，那么你还是看完吧，我已经顺利的安装完了！

假如有条件选择别的linux系统，请搜索Ubuntu 安装python3，然后可以关闭此网页

centos7本身自带有python2（yum就是python写的），所以你安装python3，是需要设置python路径，并且需要设置pip的路径和yum本身的路径

# 安装步骤-前方地狱

# 1.准备依赖

python3的安装需要一定的依赖，务必使用root权限安装（sudo）。

yum update -y
yum groupinstall -y 'Development Tools'
yum install -y gcc openssl-devel bzip2-devel libffi-devel wget
1
2
3

# 2.下载python3.9

确保执行无缺漏依赖（也就是不要error导致中断）

wget https://www.python.org/ftp/python/3.9.0/Python-3.9.0.tgz -O /tmp/Python-3.9.0.tgz
cd /tmp && tar zxf Python-3.9.0.tgz
cd Python-3.9.0
1
2
3

# 3.安装python3.9

安装是需要等一会

./configure prefix=/usr/local/python3
make && make install
export PATH=$PATH:/usr/local/python3/bin/
1
2
3

建立python软连接（也就是把python路径添加到path）

ln -s /usr/local/python3/bin/python3.9 /usr/bin/python3
ln -s /usr/local/python3/bin/pip3 /usr/bin/pip3
1
2

# 4.验证python安装是否成功

python3
# 显示已省略，反正可以看到python3.x和>>>，ctrl + D 退出
