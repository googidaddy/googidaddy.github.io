# 一、安装Docker(centos)

来自官网的力量！

  1. 安装依赖包

    sudo yum install -y yum-utils

  2. 添加yum软件源

    sudo yum-config-manager \
    --add-repo \
    https://download.docker.com/linux/centos/docker-ce.repo
    1
    2
    3

    （可选）国内源

    sudo yum-config-manager \
    --add-repo \
    https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
    1
    2
    3

  3. 安装docker核心

    sudo yum install docker-ce docker-ce-cli containerd.io

  4. 运行docker

    设置开机自启动

    sudo systemctl enable docker

    运行docker

    sudo systemctl start docker

  5. 测试docker是否安装好

    下面这个命令会拉一遍helloworld这个仓库，成功的结果大家能知道的。

    sudo docker run hello-world

# 二、实战安装Redis

个人十分建议以这种方式来安装Redis，没错！我在教你做事

  1. 下载镜像

    docker pull redis

  2. 执行镜像

    创建一个容器来执行镜像，容器名字 指定端口和背景运行

    docker run --name redis-lab -p 6379:6379 -d redis

  3. 检查容器

    docker ps

    从镜像到容器的过程.

  4. 测试redis

    经过docker ps查询到容器的id，前四位，exec方式进入容器内部，bash命令行打开

    docker exec -it e819 bash

# 三、Docker基本命令

对于初学者来说，docker用到的命令并不多，遵循着不变的启动->暂停->删除 or 恢复启动

  1. 镜像操作(包含容器)

    拉取镜像

    docker pull 某镜像

    查看镜像

    docker images

    删除镜像

    先查看目前拥有的镜像记好id

    一步一步来

    // 确定你一个容器都没有
docker rmi 镜像id
