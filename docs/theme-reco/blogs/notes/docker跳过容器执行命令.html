docker小技巧合集-持续更新

sicko 2020-12-29 docker

阅读本文大概需要1分钟 参考链接

# 利用paramiko进入了服务器上，但是进入了docker容器上是无法执行shell的命令的，我思考了好久，才知道这个问题，随后使用了docker exec -it xxx bash -c '需要执行的命令'绕过了上面的问题

# 个人不建议在docker里装载ssh并通过此途径连接，这是一种脱裤子放屁行为，应该养成良好习惯，部署到容器里的程序就不应该进入容器进行修改。

# 如何优雅的退出容器但是不停止容器运行？

ctrl+P+Q 切记，如果你有热键绑定，先取消 然后docker ps就能看到你的容器在运行
