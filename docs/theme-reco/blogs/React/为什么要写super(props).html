# 为什么要写super(props)

来源

# 一、调用父类构造函数之前不能在constructor中使用关键字this

class Checkbox extends React.Component {
  constructor(props) {
    // 🔴  还不能使用 `this`
    super(props);
    // ✅  现在可以了
    this.state = { isOn: true };
  }
  // ...
}

Javascript会强制你在使用this之前先行调用super

class Person {
  constructor(name) {
    this.name = name;
  }
}

class PolitePerson extends Person {
  constructor(name) {
    this.greetColleagues(); // 🔴  这是禁止的
    // 此时的name还没有赋值就调用了greetCollegues(),是毫无意义的
    super(name);
  }
  greetColleagues() {
    alert('Good morning folks!'+this.name);
  }
}
