# 后台自启动

后台自启动不是什么黑魔法了，只是手段比较脏。

iOS 有个 api 是后台定位的，在退出之后仍然可以注册一个服务。

然后还有另外一个 api 是可以申请一下后台任务运行时间，大概 10 分钟左右。原本是让开发者用来在耗时的任务时用户退出后可以继续处理的。

结果两个互相配合一下就可以在后台无限运行。
