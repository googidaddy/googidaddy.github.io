# 技巧:

搜索 可以加kind属性前缀，像是kind:mp3 音乐.mp3 如何使用Mac自带的tts（文字转语音）say -v Ting-ting

# m1 mac录屏

推荐这个靓仔的视频设置！obs安装brew install --cask obs 录屏肯定要录制系统内置的声音 blackhole2ch安装brew install --cask blackhole-2chobs设置-视频将两个分辨率调成关于主机-显示器-分辨率数值 obs添加显示器捕获-然后把可视窗口调整为能录到整个屏幕 command+space搜midi选择系统自带的音频设置 增加一个混合输出，48000hz，Mac主输出，勾上Mac和blackhole 最后obs增加音频输出，选择blackhole即可。
