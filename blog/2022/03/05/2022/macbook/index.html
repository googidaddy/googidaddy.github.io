配置新macbook

sicko
Live man

欣赏一下整洁的街道，图源 unsplash

软件清单:​

jetbrains 全家桶用 tool box 安装 vscode 用 brew，直接官网搜 bear 笔记 app store

配置相关:​

  1. Git brew 通过安装来更新, 系统自带 git,配置相当简单
ssh-keygen -t rsa -C “account email” ->上面的邮箱地址

然后一路回车再把公钥复制到 github 即可。 2. 系统自带 zsh 3. python brew install python 指定版本brew install python@3.9 指定版本文件在/opt/homebrew可找到 poetry brew install poetry

技巧:​

搜索 可以加 kind 属性前缀，像是 kind:mp3 音乐.mp3 如何使用 Mac 自带的 tts（文字转语音）say -v Ting-ting word 将文件保存到 OneDrive 即可开启自动保存

m1 mac 录屏​

推荐这个靓仔的视频设置！ obs安装brew install --cask obs 录屏肯定要录制系统内置的声音 blackhole2ch 安装brew install --cask blackhole-2ch obs 设置-视频将两个分辨率调成关于主机-显示器-分辨率数值 obs 添加显示器捕获-然后把可视窗口调整为能录到整个屏幕 command+space搜midi选择系统自带的音频设置 增加一个混合输出，48000hz，Mac 主输出，勾上 Mac 和 blackhole 最后 obs 增加音频输出，选择 blackhole 即可。
